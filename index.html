Vers une architeture compréhensible de l'ordinateur
Algorithme Guillemin

Présentation

Passionné par l'ordinateur, je travaille en autodidacte sur une architecture d'ordinateur "simple" à comprendre. L'idée est de penser un ordinateur fonctionnant en octal codé binaire. Les chiffres de la base 10 se code sur 4 bits, mais avec ces 4 bits on peut aussi coder 6 autres chiffres. Travailler en decimal codé binaire impliquerais des circuits électroniques non optimisés à mon sens ... La base octal (base 8) semble donc être une base intéressante pour l'optimisation des circuits combinatoires car sur 3 bits on code un maximum de chiffre contenue dans la base 10, à savoir 0, 1, 2, 3, 4, 5, 6, 7. Je vais donc essayer de créer une architecture d'ordinateur fonctionnant en octal codé binaire.
