Additionneur Guillemin

Il s'agit d'un additionneur entre deux chiffres octal codé binaire et d'une retenue codé sur 1 bit.

Tout d'abord, j'ai réalisé un programme python afin de générer la table d'addition octal pour connâitre les résultats attendues par cet additionneur.

Ensuite, j'ai réalisé un programme python qui génère les tables de vérités pour chacune des sorties de cet additionneur.

Ensuite, j'ai utilisé le logiciel QMCP pour minimiser ces fonctions logiques afin d'obtenir une optimisation de cet additionneur.

Enfin j'ai réalisé cet additionneur avec le logiciel Logisim afin de le tester.
