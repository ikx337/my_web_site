Algorithme Guillemin

Cet algorithme permet la conversion en base B ou le découpage d'un nombre en chiffre.

1/ Exemple : conversion de 137 en base 8

  • 137 >= 8**0, (8**0 = 1) ? -> vraie
  • 137 >= 8**1, (8**1 = 8) ? -> vraie
  • 137 >= 8**2, (8**2 = 64) ? -> vraie
  • 137 >= 8**3, (8**3 = 512) ? -> faux

Les poids à considérés sont donc : 64, 8, 1

  • 137/64 : quotient = 2, reste = 9
  • 9/8 : quotient = 1, reste = 1
  • 1/1 : quotient = 1, reste = 0

La jointure des quotients donne 211 et (137)10 = (211)8

2/ Exemple : découpage en chiffre du nombre 211

  • 211/100 : quotient = 2, reste = 11
  • 11/10 : quotient = 1, reste = 1
  • 1/1 : quotient = 1, reste = 0

Les quotients correspondent aux chiffres composant le nombre de départ : 211 donne 2,1,1
